# Projeto Acadêmico - Programação Orientada a Objetos II

Este repositório contém as atividades práticas desenvolvidas para a disciplina de **Programação Orientada a Objetos II**. O objetivo do projeto foi aplicar conceitos avançados da linguagem Java através de exemplos práticos de interface gráfica, concorrência, padrões de projeto e persistência de dados.

## Tecnologias e Conceitos Aplicados

### 1. Interfaces Gráficas (GUI Swing)
*   **Atividade 1:** Implementação de uma interface visual utilizando `JFrame` e o gerenciador de layout `FlowLayout`. 
*   **Fundamentos:** Uso de componentes como `JComboBox`, `JCheckBox`, `JTextField` e `JButton`, além da gestão de ciclo de vida da janela.

### 2. Programação Concorrente (Threads)
*   **Atividade 2:** Desenvolvimento de tarefas assíncronas utilizando a interface `Runnable`.
*   **Fundamentos:** Criação e execução de múltiplas threads em paralelo, controle de execução com `Thread.sleep()` e gerenciamento de estados de thread.

### 3. Design Patterns (Singleton)
*   **Atividade 3:** Aplicação do padrão de projeto criacional **Singleton**.
*   **Fundamentos:** Garantia de uma única instância de classe em memória, uso de construtores privados e métodos de acesso estáticos (Lazy Initialization).

### 4. Banco de Dados (JDBC MySQL)
*   **Atividade 4:** Conexão e manipulação de dados em um banco de dados relacional (MySQL).
*   **Fundamentos:** Uso da API JDBC para estabelecimento de conexão, execução de comandos SQL (`SELECT`) e processamento de resultados com `ResultSet`.
